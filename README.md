🛒 OldShop – Used Product Marketplace

OldShop is a simple used-product marketplace system that allows users to buy and sell second-hand products online.
This project was developed as a collaborative student project to improve practical software development skills.

## 🎥 Project Demo

[![Watch the video](https://img.youtube.com/vi/u6LGtVI7K2o/maxresdefault.jpg)](https://youtu.be/u6LGtVI7K2o)

📖 Overview

The goal of this project is to build an online platform where users can:

Browse used products

Post products for sale

Search for items

Manage product listings

Simulate a basic second-hand marketplace system

The project was initiated to practice real-world development workflows, including system design, development, and evaluation.

👥 Team

This project was developed by 5 university students working collaboratively.

Unlike many projects, we did not divide fixed roles.
All members participated in the entire development lifecycle, including:

Planning

System design

Development

Testing

Evaluation

My contribution mainly focused on:

Supporting system design

System evaluation and testing

Coordinating the project initiation
